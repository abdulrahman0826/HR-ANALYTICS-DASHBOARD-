# HR-ANALYTICS-DASHBOARD-
HR Analytics dashboard using power Bi ,showcasing total Employee & categories.
**üìä HR Analytics Dashboard:** Workforce Attrition AnalysisThis repository contains an end-to-end Human Resources (HR) Analytics solution, including the raw dataset and a fully interactive Power BI dashboard (HR_DASHBOARD.pbix). 
The project focuses on Workforce Attrition to help HR and management understand employee turnover, identify key drivers, and develop data-driven retention strategies.
**üõ†Ô∏è Project Goals**-The primary objective of this dashboard is to transform raw employee data into actionable business intelligence by answering critical questions:What is the current Overall Attrition Rate?Which Departments and Job Roles are most affected by attrition?Are factors like Salary, Job Satisfaction, Age, or Work-Life Balance significantly driving turnover?What is the demographic profile (Age Group, Gender, Marital Status) of employees who are leaving?
